# Playwright Test Automation Framework

This project is a result of my independent work as part of learning test automation with the Playwright framework. I am developing this project to refine my engineering skills in creating stable, scalable, and readable E2E tests.

## Technologies Used
* **Language:** Java 21
* **Framework:** Playwright (Java)
* **Test Engine:** JUnit 5
* **Reporting:** Allure Framework
* **Project Management:** Maven
* **CI/CD:** GitHub Actions (Automated cloud infrastructure)

## Target Websites
To expand the framework and test various scenarios, I have utilized the following platforms:
* demoqa.com
* letcode.in
* practicetestautomation.com
* saucedemo.com
* the-internet.herokuapp.com

## Key Features
* **Context Management:** Advanced handling of multiple browser tabs and windows.
* **Encapsulation:** Page Object Model (POM) pattern ensuring code readability and maintainability.
* **File Handling:** Professional implementation of file uploads and downloads, bypassing system dialogs.
* **Iframe Testing:** Seamless interaction with complex, nested iframe structures.
* **Reporting:** Automated generation of interactive Allure reports, including screenshots on test failures.
